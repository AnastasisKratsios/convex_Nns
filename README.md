# Convex Neural Functionals (CNFs)

This repository contains a small PyTorch implementation of **Convex Neural Functionals (CNFs)**: a structured neural architecture designed to preserve **convexity** and **Lipschitz regularity** by construction.

The core idea is simple: instead of training an unrestricted ReLU MLP, whose learned parameters may easily destroy convexity, we train within an admissible architecture class whose every parameter configuration remains convex and Lipschitz.

## Overview

A CNF is built from:

- finite-dimensional linear measurements of the input,
- nonnegative affine layers,
- gated PReLU activations with slopes constrained to `[0,1]`,
- blockwise max-pooling layers,
- a final nonnegative affine output layer.

These constraints ensure that every trained model remains convex and Lipschitz throughout optimization.

The repository also includes a toy experiment where a CNF is trained to approximate data generated by a larger randomly initialized convex ReLU MLP.

## Main Experiment

The experiment proceeds as follows:

1. Generate a random convex target function using a positive-weight ReLU MLP.
2. Sample finitely many training points.
3. Train a smaller CNF on these samples.
4. Verify numerically that the trained CNF satisfies Jensen convexity tests.
5. Plot the target, samples, and trained CNF approximation.
6. Save the resulting figure as a PDF.

For example, in the one-dimensional experiment, the target function is generated by a wider random convex ReLU MLP, while the CNF uses fewer trainable parameters and still preserves convexity by construction.
